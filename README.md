## Install

Inside the folder of your Hugo site, run:

```bash
$ cd themes
$ git clone https://github.com/nanxiaobei/hugo-paper
```

For more information read the official [guide](https://gohugo.io/themes/installing-and-using-themes/) of Hugo.
